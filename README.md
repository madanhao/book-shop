# book-shop
码书商店 微信小程序项目

    本项目是仿照微信小程序码书商店写的，只做学习交流，不做任何商业用途。

### 目录结构
##### book-shop-static
    book-shop-static项目是静态页面，没有与后端的交互。

##### book-shop-service
```
book-shop-service是用Java语言写的后端服务项目，框架为SpringBoot、SpringMVC、SpringDataJPA，数据库为MySQL。
```
